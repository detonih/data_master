Build image
docker build -t hadoop-base ./base

Execute bash inside container 
docker run -it --entrypoint /bin/bash hadoop-base

Stop all containers
docker container stop $(docker ps -a -q)

Inspect container
docker container inspect container_id

Find the host machine ip in the docker network.
docker inspect <container-id-or-name> | grep Gateway

Execute sql script in Dockerized database
docker exec -i data_master_db mysql -u root -p1234 < ./scripts/sql/enade_ddl.sql

List networks
docker network list

Inspect networks to look at the details
docker network inspect <id>
